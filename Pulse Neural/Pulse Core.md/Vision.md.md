# Vision

Pulse aims to become the default platform for nightlife decision-making.

---

## Short-Term Vision

- Achieve strong adoption in Galway
    
- Build a dense, active user base (300–500 users minimum)
    
- Ensure the app provides real value even without monetisation
    

---

## Mid-Term Vision

- Expand to other Irish cities (Dublin, Cork)
    
- Improve data accuracy and prediction
    
- Introduce monetisation (venue promotions, partnerships)
    

---

## Long-Term Vision

Pulse evolves into:

- A real-time nightlife intelligence platform
    
- A tool used by both users and venues
    
- A multi-city (and potentially international) product
    

---

## Strategic Position

Pulse is not just:

- A map
    
- A social app
    

It is:

> A decision engine for nightlife

---

## End Goal

When users think:  
"Where should we go tonight?"

Pulse is the first thing they open.
