Pulse is a nightlife discovery app focused on students (initially Galway).

Core Value:
- Shows where people are going BEFORE you go out
- Helps avoid dead nights
- Creates confidence in going out

Key Hook:
"See where everyone is going tonight"

Target:
- Students (NUIG, ATU)
- Social groups
- Nightlife-heavy users

Core Loop:
1. User opens app
2. Sees live / planned crowd data
3. Decides where to go
4. Pulses (adds signal)
5. Data improves
6. Others open app
